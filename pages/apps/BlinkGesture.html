Overview

BlinkGesture is a desktop tool that lets you control your PC using eye blinks. Powered by Python, it uses OpenCV and dlib‚Äôs facial landmark detection to calculate Eye Aspect Ratio (EAR) and maps left/right blinks to keyboard actions like arrow keys or other inputs. Great for hands-free navigation, accessibility, or experimenting with computer vision.

How It Works

  • Uses OpenCV to access webcam
  • Detects facial landmarks via dlib
  • Computes EAR in real-time
  • Blinks mapped to keyboard actions using pyautogui
